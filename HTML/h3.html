现在请在你的 HTML 文档的body中输入： 

			<h1>我的第一个标题</h1>
			<p>我的第一个段落。</p>


看看出现了什么？


在<h1>标签里出现了一段黑色粗体的文字
通常作为 HTML 文档里的标题

在<p>标签里出现了一段黑色的普通文字
通常作为 HTML 文档里的段落

至于为什么h1与p会有 空隙 并且 竖直排列
我们将会在后面的章节提到


将下列两个示例代码复制到编辑器里，看看出现什么效果吧
		<html>
		<head>
			<title>
				我的第一个网页
			</title>
		</head>
		<body>
			<h1>h1对应的标题</h1>
			<h2>h2对应的标题</h2>
			<h3>h3对应的标题</h3>
			<h4>h4对应的标题</h4>
			<h5>h5对应的标题</h5>
			<h6>h6对应的标题</h6>
		</body>
		</html>
		


		<html>
		<head>
			<title>
				我的第一个网页
			</title>
		</head>
		<body>
			<p1>p1对应的段落</p1>
			<p2>p2对应的段落</p2>
		</body>
		</html>
