CSS作为修饰样式的工具
在为元素添加样式的时候
有3个要素不能忽略
分别是：选择器、属性和值

选择器是用来选取要修饰的对象
属性是指要修饰的内容
值对应的就是修饰的成果


在前面的例子中
CSS将body中的文本变成了红色






也自己试试看其他的吧！


将下列两个示例代码复制到上方的在线编辑器里，看看出现什么效果吧
		<html>
		<head>
			<title>
				我的第一个网页CSS
			</title>
		</head>
		<body>
			<p>我想要变成rgb（110，91，115）！</p>
		</body>
		</html>
		
		p{
			color: rgb(110,91,115);
		}
